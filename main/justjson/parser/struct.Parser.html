pub struct Parser<'a, const GUARANTEED_UTF8: bool> { /* private fields */ }
Expand description

Parses JSON, driven by a ParseDelegate.

This is a low-level type. In general, users will want to use either the Document or Value types instead of directly interacting with the parser.

This type uses a constant to track whether UTF-8 needs to be manually verified or not. This allows the compiler to optimize the &[u8] and &str parsing methods independently.
