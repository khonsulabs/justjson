pub enum Value<'a> {
    Number(JsonNumber<'a>),
    String(JsonString<'a>),
    Boolean(bool),
    Object(Object<'a>),
    Array(Vec<Value<'a>>),
    Null,
}
Available on crate feature alloc only.
Expand description

A JSON value.

The Backing generic is the storage mechanism used by JsonNumber and JsonString. This is generally &str or Cow<str>.
